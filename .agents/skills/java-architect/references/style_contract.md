# Style Contract

## Typography
- **Body Text**: Serif font (Georgia, Lora, Cambria), 10-11pt, 1.2-1.5 line spacing, justified.
- **Headings**: Sans-serif font (OpenSans, Calibri, Arial), bold, dark color (#1a1a2e or #1a1a1a).
  - H1: 22-32pt
  - H2: 16-22pt
  - H3: 13-16pt
- **Code**: Monospace font (Consolas, Inconsolata), 9-10pt, light gray/blue background (#f4f6f8).

## Color Palette
- **Primary text**: #1a1a1a or #1a1a2e
- **Secondary text**: #4a4a4a
- **Accent**: Muted dark gray or subtle accent (red/orange)
- **Code background**: #f4f6f8

## Page Layout
- **Margins**: 1 inch (2.54cm) top/bottom/right, 1.2 inch (3cm) left (gutter).
- **Headers/Footers**: Chapter name and page number.
- **No intentionally blank pages** between sections (except part dividers).

## Cover Page Design
- **Background**: Dark charcoal or deep navy solid color (no gradients).
- **Title**: Large bold white sans-serif.
- **Visual Element**: Subtle geometric pattern, architectural photograph, or abstract illustration in muted tones.
- **No-Nos**: No light backgrounds, no neon colors, no simple gradient backgrounds.

## Code Block Formatting (Critical)
1. **Each statement on its own line**.
2. **Proper indentation**: 4 spaces.
3. **Background**: Light gray/blue shading behind the entire code block.
4. **Spacing**: 6pt padding before and after.
